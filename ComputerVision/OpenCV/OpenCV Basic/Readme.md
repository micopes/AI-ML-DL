#### matplot.py
- matplotlib 이용 이미지 출력
- cv2.imshow()로 이미지를 받으면 BGR인 것을 RGB로 바꿔서 출력해줘야 한다.
- cmap = 'gray'를 통해 grayscale 이미지 출력

#### camera_in
- edge 검출
- 동영상 출력

#### Video_in
- edge 검출
